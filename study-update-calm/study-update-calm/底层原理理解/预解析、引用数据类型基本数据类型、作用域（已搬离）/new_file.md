
## 第一步变量提升、函数名提升
```js
var a;
var fn;
```

## 第二步按照代码书写顺序执行,遇到函数调用，进入函数作用域
```js
a = [1];
fn = function(a){
	...
}
fn(a);
```

## 第三步,函数内部
```js
	var a;
	a = window.a = [1];//数组是引用数据类型，保持引用关系
	//实际的数组元素被存储在堆（heap）内存中；数组引用变量是一个引用类型的变量，被存储在栈（stack）内存中
	console.log(a,'+++');
	a[1] = 10;//对数组操作，会根据引用关系，操作数组存放’数据栈‘的内存地址
	a = 2;//给其赋值一个基本数据类型，打破了引用关系。
	console.log(a,'---')
```
