题目描述

输入一个大于0的十进制数,将其转换为二进制

解题思路

10进制转2进制实际上就是用10不断去除以2,将余数从下往上返回即可

        const ten2Two = function(number) {
            console.log(number);
            // 声明一个栈
            const stack = new Stack();
            // 声明一个字符串
            let binary = "";
            // 入栈
            while (number > 0) {
                // 取余入栈
                stack.push(number % 2);
                // 取整
                number = Math.floor(number / 2);
            }
            // 出栈
            while (stack.size() > 0) {
                // 拼接字符串
                binary += stack.pop();
            }
            return binary;
        };
