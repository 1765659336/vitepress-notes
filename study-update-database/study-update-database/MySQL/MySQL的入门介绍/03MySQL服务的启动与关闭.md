## 命令行启动关闭MySQL服务

管理员的身份启动cmd

![image-20210604174330349](03MySQL服务的启动与关闭.assets/image-20210604174330349.png)

启动语句

```mysql
net start MySQL
```

关闭语句

```mysql
net stop MySQL
```
